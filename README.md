# Color

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple JavaScript library for color manipulation.

## Features
- Conversion between hex color codes and RGB values
- Conversion between RGB and HSV color models
- Lightening of colors based on a given ratio

## Usage

To use the library, import the `Color` class from the JavaScript file:

```JavaScript
import { Color } from "https://code4fukui.github.io/Color/Color.js";

console.log(Color.color2rgb("#f88"));
```
