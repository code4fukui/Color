# Color

色操作のためのシンプルなJavaScriptライブラリ。

## 機能
- 16進数カラーコードとRGB値の相互変換
- RGBとHSV色モデルの相互変換
- 指定された比率に基づく色の明度調整

## 使い方

ライブラリを使用するには、JavaScriptファイルから`Color`クラスをインポートします:

```JavaScript
import { Color } from "https://code4fukui.github.io/Color/Color.js";

console.log(Color.color2rgb("#f88"));
```
